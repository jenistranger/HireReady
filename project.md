# Resume Tailor

ИИ-таилоринг резюме. Вставь резюме, вставь описание вакансии — получи переписанное резюме под эту роль без единой выдуманной строки.

FastAPI + vanilla JS. Работает в Docker. Без базы данных, без аккаунтов.

## Описание

Resume Tailor берёт резюме и описание вакансии, и переписывает резюме под эту роль — меняет порядок разделов, подставляет терминологию из вакансии, выделяет нужный опыт. Ничего не выдумывает: системный промпт явно запрещает добавлять навыки, сертификаты или опыт, которых не было в оригинале.

### Как работает

1. Пользователь вставляет текст резюме (или загружает PDF)
2. Вставляет описание вакансии (или PDF / URL)
3. Бэкенд отправляет оба текста в OpenRouter со структурированным промптом
4. ИИ возвращает адаптированное резюме по 7 строгим правилам
5. Пользователь копирует результат, скачивает PDF или открывает в полноэкранном режиме

### PDF-пайплайн

- **Входной PDF:** PDF → pypdf → текст → в ИИ
- **URL вакансии:** URL → httpx → HTML stripped → обрезка до 5000 символов
- **Выходной PDF:** текст от ИИ → fpdf2 → структурированный PDF с тёмно-синей шапкой, разделителями секций, буллит-пунктами, формат A4

### Правила системного промпта (7 штук)

1. Не выдумывать факты
2. Органично встраивать ключевые слова из вакансии
3. Переставить разделы опыта по релевантности
4. Использовать профессиональный язык под конкретную роль
5. Сохранять логику структуры оригинала
6. Язык вывода = язык оригинального резюме
7. Только текст резюме — без вводных фраз и комментариев

## Дорожная карта

### Текущее состояние (v1.3.0)
Основной цикл работает end-to-end: резюме на входе → адаптированное резюме → скачать PDF. Stateless, контейнеризован, готов к запуску локально или на VPS.
